# Topic-Modeling
This repository contains the codes for unspuervised document classification. The code takes in the documents along with their IDs and returns the Document ID with the corresponding topic IDs. It also generates the list of words under each topic. The following dependencies are to be installed before running the script. \br
1. spacy==2.0.16
2. nltk==3.2.5
3. gensim==3.4.0

The code may work with different versions of the above mentioned packages, but the functionalities might differ.
